# dart-cli-crypto-scrypt

## Description
Comparing passwords with one-way hashes.

## Tech stack
- dart
  - steel_crypt

## Docker stack
- google/dart

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
